About the Project

This project focuses on the application of radiomic analysis to lung cancer images, aiming to improve diagnosis and treatment outcomes through advanced image processing techniques.

Project Details

Our approach involves using advanced radiomic features to analyze lung cancer images. We utilize various machine learning algorithms to classify and predict patient outcomes based on these features.
